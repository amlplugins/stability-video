# @amlplugins/stability-video

Native `stability-ai` SDK re-exported under the `@amlplugins` namespace.

> Stability AI Video — Image-to-Video diffusion generation with motion-bucket steering.

## Install

```bash
npm install @amlplugins/stability-video
```

## Use

```ts
import { /* named exports from stability-ai */ } from "@amlplugins/stability-video";
// or
import sdk from "@amlplugins/stability-video";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
